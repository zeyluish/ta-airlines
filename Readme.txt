## Installation
Clonez le repository : https://github.com/zeyluish/ta-airlines

##Utilisation
Après avoir lancé le projet, il est possible de  :
- Créer un avion, modifier ses attributs, lui affecter un vol et vérifier qu'il est disponible aux bons horaires
- Créer un vol, modifier ses attributs, faire un planning de vol selon le jour, annuler un vol et faire une liste des passagers
- Créer un aéroport, modifier ses attributs, attribuer à un vol un aéroport de départ de d'arrivé
- Confirmer une réservation, la modifier ou l'annuler
- Affecter un pilote à un vol
- Affecter le personnel cabine à un vol
- Un passager peut réserver un vol, l'annuler ou obtenir les informations sur sa réservation

##Fonctions particulières
Pour utiliser les fonctions modifier, par exemple modifierAvion, il faut préciser l'attribut que l'on souhaite modifier puis la valeur. Les deux attributs doivent être en string
